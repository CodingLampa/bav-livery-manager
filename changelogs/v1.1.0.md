## Changelog v1.1.0

## Features:
- White theme — now you can choose between the dark and light themes in the settings
- Reworked settings page
- Improved filters — now no need to expand it, filter liveries in a single click
- Improved search now it is smarter and works better with typos and partial matches
- Continuous scrolling — no more need to switch between pages, just scroll down to load more liveries
- Various UI improvements and optimizations

### Bug Fixes:
- Fixed the tour not showing pictures and videos

## Roadmap:
- Integrated SpeedCore support, allowing required packages to be installed automatically alongside compatible liveries
