## Changelog v0.3.3

### Features:
- Added a cancellation button that allows users to cancel downloads
- Replaced My Downloads page with Updates page
- Various changes to interface (thanks to John Yang for feedback)
- Added an introduction tour of the manager for new users
- Added custom title bar
- Added new taskbar icon
- SpeedCore error message displays when downloading liveries that depend on it

### Bug Fixes:
- Fixed PMDG livery installation issue causing liveries not to show in Operations Center 3
- PMDG liveries will now install successfully if you have not previously installed one via Operations Center 3

## Roadmap:
- Detailed livery information pages featuring airframe data, a full changelog for each livery, and quick access to aviation resources including FR24, AirNavRadar, and Planespotters
- Integrated SpeedCore support, allowing required packages to be installed automatically alongside compatible liveries
